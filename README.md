# 벡터(SVG) 기반의 드로잉 툴 구현


### 사용 기술
- React
- TypeScript
- Vite
- [Konva.js](https://konvajs.org/)
- [Konva React](https://konvajs.org/docs/react/Intro.html)


### 구현 내용

- 드로잉 타입 선택
  - 직선
  - 곡선 (자유로운 선 드로잉)
  - 타원 (shift 키 조합 시 원)
  - 직사각형 (shift 키 조합 시 정사각형)
  - 다각형 (3 ~ 9각형까지 허용)

- 드로잉 타입 키보드로 선택 가능

- 선 두께/컬러 선택
  - 선 두께는 5px ~ 50px
    - input type="range" 로 간단히 구현
  - 컬러는 input type="color" 로 간단히 구현

- Undo, Redo
  - 최근 40개의 작업 기록만 저장

- Transformer
  - 선택한 도형의 크기/각도 조절




### 회고
